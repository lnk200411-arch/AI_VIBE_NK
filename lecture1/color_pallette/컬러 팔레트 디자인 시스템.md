## 컬러 사용 가이드

### Primary — 라임 그린 (`#72E040`)
- 페이지에서 가장 넓은 면적을 차지하는 핵심 브랜드 컬러
- 히어로 섹션 배경, 강조 카드 배경, 섹션 구분 블록에 적용
- 라임 그린 위에는 반드시 흰색(`#FFFFFF`)만 텍스트로 사용해 가독성 확보
- 그라데이션 사용 시 권장 방향: `#A8F07A → #72E040 → #50B828`

### Accent — 코랄 레드 (`#E84228`)
- 제목, 강조 문구, CTA 버튼에 집중 사용
- 라임 그린과 보색 관계 — 함께 배치 시 강렬한 시각적 에너지 연출
- 크림 배경(`#FFFAF5`) 위에서 주요 정보 텍스트로 활용
- 남용 금지 — 시선을 끌어야 할 핵심 요소에만 배치

### Secondary — 크림 화이트 (`#FFFAF5`)
- 페이지 기본 배경 및 여백 색상
- 콘텐츠 섹션 간 숨 고르는 중립 공간에 적용
- 라임 그린 섹션과 자연스럽게 대비를 이루며 시각적 리듬 형성

### Decorative — 피치 오렌지 (`#F0956A`)
- 일러스트, 아이콘, 장식 요소에만 제한 사용
- 그린·레드 조합의 강렬함에 온기와 부드러움 추가하는 역할
- 본문 텍스트에는 사용 금지 (WCAG 접근성 대비 기준 미달)

### 컬러 사용 비율 가이드
| 색상 | 권장 비율 | 역할 |
|------|-----------|------|
| `#72E040` 라임 그린 | ~60% | 메인 배경, 핵심 브랜드 영역 |
| `#FFFAF5` 크림 화이트 | ~15% | 외부 배경, 여백 |
| `#FFFFFF` 순백 | ~15% | 그린 위 타이포그래피, 아이콘 |
| `#E84228` 코랄 레드 | ~8% | 제목, 버튼, 장식 일러스트 |
| `#F0956A` 피치 오렌지 | ~2% | 일러스트 포인트 장식 |

## 반응형 고려사항

### 다크모드 대응

```css
@media (prefers-color-scheme: dark) {
  :root {
    /* 배경: 딥 다크 그린 계열로 전환 */
    --color-bg-primary: #0F1F08;
    --color-bg-secondary: #1A3D0A;

    /* 텍스트: 라임 계열로 반전 */
    --color-text-primary: #A8F07A;
    --color-text-secondary: #FFFAF5;
    --color-text-muted: #F5C8A0;

    /* Primary: 밝기 높여 다크 배경 위 대비 확보 */
    --color-primary: #A8F07A;
    --color-primary-dark: #72E040;
    --color-primary-light: #D4F7B8;

    /* Accent: 코랄 레드 밝기 조정 */
    --color-accent: #FF6B52;
    --color-button-primary: #FF6B52;
    --color-button-hover: #E84228;

    /* Link */
    --color-link: #A8F07A;
    --color-link-hover: #72E040;

    /* Border */
    --color-border-default: #2A4A1A;
    --color-border-active: #FF6B52;
  }
}
```

### 화면 크기별 고려사항

- **모바일 (xs ~ sm)**: 라임 그린 배경 비중을 줄이고 크림 배경 여백 확보. 버튼·링크 터치 영역 최소 48px 이상 유지.
- **태블릿 (md)**: 그린 카드와 크림 카드를 교차 배치해 섹션 간 리듬감 연출.
- **데스크톱 (lg ~ xl)**: 히어로 섹션 라임 그린 전폭 배경 적용. `maxWidth` 컨테이너 밖 여백은 크림 배경으로 자연스럽게 처리.

### 접근성 (WCAG AA 기준)

| 조합 | 대비율 | 기준 충족 |
|------|--------|-----------|
| `#FFFFFF` (텍스트) + `#72E040` (그린 배경) | 3.8:1 | AA 대형 텍스트만 충족 |
| `#E84228` (텍스트) + `#FFFAF5` (크림 배경) | 4.7:1 | AA 충족 |
| `#FFFFFF` (텍스트) + `#E84228` (레드 버튼) | 4.2:1 | AA 충족 |
| `#FFFFFF` (텍스트) + `#C43020` (레드 호버) | 5.1:1 | AA 충족 |

> ⚠️ 주의: 라임 그린 배경 위 일반 본문 텍스트는 AA 기준(4.5:1) 미달. 본문에는 흰색 텍스트를 대형 폰트(18px 이상)로 사용하거나, 배경을 `#50B828` 이하 어두운 그린으로 변경 권장.

## 디자인 특징 요약

**전체 분위기**: 핸드메이드 감성의 **여름 인디 레트로**

- **보색 에너지**: 라임 그린 + 코랄 레드의 보색 대비로 강렬하고 생동감 있는 시각 경험 연출
- **DIY 아날로그 감성**: 크레파스 질감 그린 + 손그림 일러스트가 디지털 환경에서도 핸드드로잉 느낌을 살림
- **여름 무드**: 라임 그린(싱그러움) + 피치 오렌지(달콤함) + 코랄 레드(생동감) 조합으로 여름 과일처럼 풍성한 무드 형성
- **단순하고 강한 팔레트**: 3색(그린·레드·화이트) 중심 구조로 정보 위계를 명확히 처리하면서 강한 인상 유지
- **브랜드 방향성**: 과도한 세련됨보다 진정성 있는 자유로움 — 독립 음악·아트씬의 DIY 정신 반영

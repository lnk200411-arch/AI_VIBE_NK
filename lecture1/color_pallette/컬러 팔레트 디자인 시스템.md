## 컬러 사용 가이드

### Primary Color — `#7EE85A` 라임 그린
- **사용처**: 히어로 섹션 배경, 카드 배경, 섹션 구분 블록, 호버 오버레이
- **주의**: 흰색 텍스트(`#FFFFFF`)와 함께 사용 시 대비 최적. 코랄 레드와 보색 조합으로 에너지감 극대화
- **금지**: 작은 텍스트 색상으로 직접 사용 금지 (가독성 저하)

### Accent Color — `#E8502A` 코랄 레드
- **사용처**: 페이지 제목(H1/H2), CTA 버튼, 링크, 포인트 아이콘, 날짜·위치 등 핵심 정보
- **주의**: 크림 화이트(`#F5F0E8`) 또는 라임 그린(`#7EE85A`) 배경 위에서 최고의 가시성 발휘
- **활용**: `--color-accent-dark: #C03A18`은 버튼 호버, 다크 모드 강조에 사용

### Background — `#F5F0E8` 크림 화이트
- **사용처**: 전체 페이지 기본 배경, 텍스트 영역, 푸터 배경
- **특징**: 순백 대신 크림 톤을 사용해 눈의 피로를 줄이고 빈티지 감성 유지

### Text Colors
- **제목 텍스트**: `--color-text-primary: #E8502A` — 임팩트 있는 헤딩에 사용
- **본문 텍스트**: `--color-text-on-light: #3A2A1A` — 크림 배경 위 읽기 편한 다크 브라운
- **그린 배경 위 텍스트**: `--color-text-secondary: #FFFFFF` — 라임 그린 배경 영역에서만 사용

### Interactive Elements
- **Primary 버튼**: 배경 `#E8502A` / 텍스트 `#FFFFFF` / 호버 `#C03A18`
- **Ghost 버튼**: 배경 투명 / 테두리 `#E8502A` / 텍스트 `#E8502A`
- **링크**: 기본 `#E8502A` → 호버 시 `#7EE85A` (코랄→그린 전환으로 생동감 표현)

## 반응형 고려사항

### 다크 모드 적용

```css
@media (prefers-color-scheme: dark) {
  :root {
    /* 배경 반전 — 짙은 올리브 그린 계열로 대체 */
    --color-bg-primary: #1A2A12;
    --color-bg-secondary: #2E4A20;
    --color-bg-card: #243A18;

    /* 텍스트 — 크림 화이트로 역전 */
    --color-text-primary: #F5F0E8;
    --color-text-on-light: #E8DDD0;
    --color-text-muted: #8A9A7A;

    /* 라임 그린은 채도 낮춰 눈부심 방지 */
    --color-primary: #5AC438;
    --color-primary-light: #7EE85A;
    --color-primary-dark: #3A9A20;

    /* 코랄 레드는 밝게 조정 */
    --color-accent: #F07A56;
    --color-button-primary: #F07A56;
    --color-link: #F07A56;

    /* 테두리 어둡게 */
    --color-border-default: #3A4A2A;
  }
}
```

### 화면 크기별 색상 활용

| 화면 크기 | 배경 전략 | 텍스트 전략 |
|-----------|-----------|-------------|
| **모바일** (xs~sm) | 라임 그린 배경 섹션 비율 줄이기 (시각 피로 방지) | 폰트 크기 축소 시 `#E8502A` 대신 `#3A2A1A` 사용 권장 |
| **태블릿** (md) | 배경 그린/크림 50:50 분할 레이아웃 | 제목은 `#E8502A`, 본문은 `#3A2A1A` |
| **데스크톱** (lg+) | 좌우 패널 분리 (크림 ↔ 그린) | 풀사이즈 히어로에서 흰 텍스트 대형 활용 |

### 접근성 (WCAG 대비율)

| 조합 | 대비율 | 등급 |
|------|--------|------|
| `#E8502A` on `#F5F0E8` | 4.6:1 | AA ✅ |
| `#FFFFFF` on `#7EE85A` | 3.1:1 | AA (대형 텍스트만) ⚠️ |
| `#FFFFFF` on `#E8502A` | 4.5:1 | AA ✅ |
| `#3A2A1A` on `#F5F0E8` | 9.8:1 | AAA ✅ |

> ⚠️ **주의**: 라임 그린(`#7EE85A`) 위 흰색 텍스트는 18px 이상 굵은 폰트(Bold)에서만 사용할 것
